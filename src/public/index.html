Abysmal API

/nsfw

  • Returns a NSFW image
  • Query category required
  • Available query values: waifu, neko, trap, blowjob
  • Example: https://api.abysmal.eu.org/nsfw?category=waifu

/catboy

  • Returns a catboy image (SFW)
  • Example: https://api.abysmal.eu.org/catboy

/neko

  • Returns a neko image (SFW)
  • Example: https://api.abysmal.eu.org/neko
